Description

Implémentation Python d'un outil de chiffrement de fichiers basé sur la fonction de dérivation de clé Scrypt et le chiffrement symétrique Fernet. Le programme supporte le chiffrement récursif de dossiers avec gestion de sel cryptographique. Conçu pour la recherche en sécurité et l'apprentissage de la cryptographie appliquée.

Fonctionnalités

  • Chiffrement et déchiffrement de fichiers individuels
  • Chiffrement récursif de dossiers entiers
  • Gestion automatique du sel cryptographique
  • Dérivation de clé via Scrypt avec paramètres configurables

Installation

pip install cryptography
python main.py --help

Utilisation

# Chiffrer un fichier
python main.py encrypt --input secret.txt --output secret.enc

# Chiffrer un dossier récursivement
python main.py encrypt --input ./dossier/ --recursive

# Déchiffrer
python main.py decrypt --input secret.enc --output secret.txt

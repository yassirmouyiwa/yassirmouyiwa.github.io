# My Writeups — Standalone SPA

This is the **standalone Single Page Application (SPA)** for hosting all CTF writeups, vulnerability analyses, and security research.

## Features

- **Modern SPA** with client-side routing
- **Category filtering** (CTF, Web, Pwn, Reverse, Forensics, Crypto, Network)
- **Bilingual** (FR/EN) with language toggle
- **Dark/Light** theme toggle
- **Responsive** design for all devices
- **Search** functionality (coming soon)

## Structure

```
site/writeups/
├── index.html          # Main SPA page
├── style.css           # Stylesheet
├── app.js              # Application logic & routing
└── README.md           # This file
```

## Usage

This subdirectory is designed to be **servable on its own** as a complete site under `/writeups/` on the main domain.

## Stack

- Vanilla HTML5 / CSS3
- Vanilla JavaScript (no frameworks)
